## Agent 技能配置

### Issue 追踪器

通过 `gh` CLI 操作 GitHub Issues。详见 `docs/09_研发协作/issue-tracker.md`。

### Triage 标签

默认标签：`needs-triage`（待评估）、`needs-info`（待补充信息）、`ready-for-agent`（Agent 可执行）、`ready-for-human`（需人工处理）、`wontfix`（不处理）。详见 `docs/09_研发协作/triage-labels.md`。

### 领域文档

单上下文模式：`CONTEXT.md` + `docs/adr/` 放在项目根目录。详见 `docs/09_研发协作/domain.md`。
