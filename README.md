# 申论练习平台

一个面向申论训练场景的开源 AI 学习平台。

> 文档已统一整理到 [文档索引](docs/文档索引.md)。

## 项目简介

申论练习平台聚焦申论题练习与写作反馈场景，提供登录注册、邮箱验证码、个人配置、练习分析、提纲生成、答案批改、批改结果查询等能力，目标是把申论训练从单次问答扩展为可持续的练习闭环。

## 当前能力

### 后端

- FastAPI 服务与统一路由
- 用户注册、登录、找回密码、当前用户信息
- 邮件验证码发送与校验
- AI 配置管理与系统默认配置
- 题库查询、题目详情、题目导入、题目管理接口
- 练习分析、提纲生成、答案批改接口
- 批改结果列表与详情查询
- 批改链步骤记录、题目分析与批改结果持久化
- 管理员用户与邮件配置接口

### 前端

- 首页、题库页、题目详情页、练习页、批改报告页
- 登录 / 注册 / 找回密码流程
- 个人中心、AI 配置、历史记录、管理员页面
- 受保护路由、布局分层、本地草稿与示例报告
- 统一接口封装、演示题库与本地复盘闭环

## 目录边界

仓库当前已经从“顶层共享模型”演进到“模块内聚 + 工作流编排 + AI 能力拆分”的结构，详细约定见：

- [目录结构说明](docs/02_架构与目录/DIRECTORY_STRUCTURE.md)
- [技术架构说明](docs/02_架构与目录/ARCHITECTURE.md)

后端当前的核心边界是：

- `core/`：全局基础能力
- `infra/`：外部基础设施接入
- `shared/`：跨模块共享内容
- `modules/`：业务域主承载层
- `workflows/`：多步骤流程编排
- `ai/capabilities/`：细粒度 AI 能力

## 技术栈

- 前端：Vue 3、TypeScript、Vite
- 后端：Python、FastAPI、SQLAlchemy
- 模型编排：LangChain、LangChain OpenAI
- 数据库：MySQL
