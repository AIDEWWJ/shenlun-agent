## 角色说明

### 1. 访客

**定位：** 未登录或仅浏览状态用户。

**可做：**

- 查看公开介绍内容
- 了解产品功能
- 访问公开演示

**不可做：**

- 保存个人配置
- 使用完整训练流程
- 查看历史记录
- 访问私有数据

### 2. 学员

**定位：** 核心使用者。

**可做：**

- 使用申论训练全流程
- 配置自己的 AI 模型
- 记录练习、复盘与错题
- 管理自己的 Prompt 模板

**不可做：**

- 访问他人数据
- 修改全局系统配置
- 管理用户与权限
- 操作题库后台

### 3. 管理员

**定位：** 系统维护者。

**可做：**

- 管理用户和权限
- 维护全局模型配置
- 管理题库、模板和系统参数
- 查看系统状态与日志

**边界：**

- 只处理管理职责
- 不默认查看普通用户练习内容
- 访问敏感数据需遵循审计与授权机制

## 建议落地方式

- 认证：登录态 / Token
- 授权：RBAC
- 数据隔离：学员仅能访问自己的业务数据
- 审计：管理员敏感操作需记录日志
